# 📈 台股強勢起漲點量化回測系統 (Taiwan Stock Quantitative Backtesting)

本專案是一個基於 Python 與 `yfinance` 構建的台股量化交易回測框架。策略核心旨在捕捉「具備動能支撐且尚未嚴重乖離」的強勢股起漲點。透過整合長線趨勢、短線乖離、量價關係（周轉率）與動能指標（MACD、KD），打造一套兼具爆發力與防禦性的波段策略。

目前測試標的涵蓋 27 檔精選電子與電機族群個股（如：國巨、華城、中興電、東元等）。

---

## 💡 策略邏輯 (Strategy Logic)

本策略採用「多條件嚴格篩選進場，複合條件彈性出場」的波段操作思維。

### 📌 進場條件 (Entry Signals)
系統必須在同一日**完全滿足**以下 4 大技術面與籌碼面條件，才會觸發買進訊號：

1. **量價配合 (Turnover Rate)**
   * 當日周轉率介於 **2% ~ 10%** 之間（確保個股具備一定的市場關注度與流動性，排除冷門股或爆量出貨疑慮）。
2. **長中線多頭排列 (Long-Term Trend)**
   * **20 日均線 > 60 日均線**（月線大於季線，確認中期趨勢向上）。
   * **當日收盤價 > 60 日均線**（股價位於季線之上）。
3. **周線動能強勢 (Weekly MACD)**
   * **周 MACD 三指標齊揚**：周線級別的 `DIF`、`MACD` (訊號線) 與 `OSC` (柱狀體) 皆必須大於 0。
   * *⚠️ 技術亮點：為避免量化回測常見的「未來函數 (Look-ahead bias)」，本系統在計算周線指標時嚴格使用 `shift(1)`，確保日線訊號只參考「上周已確立」的周 MACD 數據。*

### 🚪 出場條件 (Exit Signals)
進場持有後，一旦觸發以下任一條件即刻平倉結算：

1. **達標停利 (Take Profit)**：未實現報酬率達 **+30%**。
2. **趨勢破壞 (Stop Loss / Trailing Stop)**：當日收盤價跌破 20 日均線達 **20%**（乖離率 `Bias20 <= -20%`）。
3. **強制平倉 (Force Close)**：回測資料結束日若仍持倉，則以最後一日收盤價強制結算。

## 🛠️ 技術架構與特色 (Features)

* **純淨歷史數據**：強制關閉 `yfinance` 的自動除權息還原 (`auto_adjust=False`)，確保技術指標（如乖離率、均線突破）是基於當時交易者眼中的**原始歷史收盤價**進行精準運算。
* **無未來函數**：周線與日線資料精準對齊，避免訊號偷看未來。
* **動態周轉率推算**：整合 Yahoo Finance API 獲取即時流通股數 (`sharesOutstanding`)，動態還原歷史日周轉率。
* **自動化績效彙整**：批次執行多檔股票回測後，自動產生包含交易次數、勝率、平均報酬與出場原因分析的彙總報表，並依勝率自動排序。
